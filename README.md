# Reset Your Aussie - Waitlist Landing Page

Waitlist landing page for Reset Your Aussie membership by Drop The Leash Dog Training.

## About

This landing page promotes the Reset Your Aussie monthly membership program for overwhelmed Australian Shepherd owners. Built with vanilla HTML/CSS and integrated with GoHighLevel form system.

## Features

- Responsive design (mobile, tablet, desktop)
- Editorial typography with Crimson Pro and Inter fonts
- Embedded GHL waitlist form
- Performance-optimized single-page design
- Science-based messaging aligned with Laura Hunter's training philosophy

## Deployment

Deployed on Vercel with automatic deployment from GitHub.
